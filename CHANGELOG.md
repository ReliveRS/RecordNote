# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [No Publicado]

### Añadido
- Widget para inicio rápido de grabación
- Compartir notas vía redes sociales

### Cambiado
- Mejora en rendimiento de transcripción
- UI actualizada de configuración

## [1.0.0] - 2025-01-15

### Añadido
- Grabación de audio de alta calidad
- Transcripción automática de voz a texto
- Gestión completa de notas
- Sistema de etiquetas
- Colores personalizables
- Sistema de favoritos
- Sincronización automática
- Backup local automático
- Exportación en múltiples formatos
- Tema claro y oscuro
- Material Design 3
- Búsqueda y filtros avanzados

### Seguridad
- Encriptación de datos sensibles
- Autenticación segura con JWT

## [0.9.0-beta] - 2024-12-01

### Añadido
- Versión beta inicial
- Funcionalidades básicas de grabación
- UI básica con Compose

### Problemas Conocidos
- Transcripción solo en español
- Sin sincronización en la nube
